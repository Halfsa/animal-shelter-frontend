# Telepítési útmutató:
klónozza a github repot:
```
  git clone https://github.com/Halfsa/animal-shelter-frontend.git
```
nyissa meg a projektet és futtassa a következő parancsot:
```
  npm install
```
majd futtassa a frontend applikációt:
```
  vite
```
